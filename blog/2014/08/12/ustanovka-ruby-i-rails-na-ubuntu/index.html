Установка Ruby и Rails на Ubuntu

Aug 12th, 2014 | Comments

Мой первый пост в этом блоге я посвящу самым азам. В этом гайде будет описана установка руби и рельс на убунту.

Присаживайтесь поудобнее, процесс установки займет какое-то время.

1) Установка Ruby

sudo apt-get update
sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev
libreadline-dev
 libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev
python-software-properties

cd
git clone git://github.com/sstephenson/rbenv.git .rbenv
echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
exec $SHELL

git clone git://github.com/sstephenson/ruby-build.git
~/.rbenv/plugins/ruby-build
echo 'export PATH="$HOME/.rbenv/plugins/ruby-build/bin:$PATH"' >> ~/.bashrc
exec $SHELL

rbenv install 2.1.2
rbenv global 2.1.2
ruby -v

echo "gem: --no-ri --no-rdoc" > ~/.gemrc

2) Установка Rails

sudo apt-get install nodejs

gem install rails
rbenv rehash
rails -v
gem install bundler

3) Запуск Rails приложение

Теперь, когда ruby и rails установлены, мы можем проверить их работоспособность, создав и запустив приложение.

rails new_app
cd new_app
rails server

4) Установка Postgres (Опционально)

sudo apt-get install postgresql libpq-dev
sudo su - postgres
createuser [me]
createdb -O [me] [название_дб]_development
createdb -O [me] [название_дб]_test
exit
